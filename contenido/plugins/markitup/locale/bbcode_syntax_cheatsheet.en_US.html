[b]bold[/b]
[u]underlined[/u]

Headers 
[header=1]Header 1[/header]
[header=2]Header 2[/header]
[header=3]Header 3[/header]
[header=4]Header 4[/header]
[header=5]Header 5[/header]
[header=6]Header 6[/header]

Note: Use allways a empty line between header and next element like paragraph or header.

Lists

Bulleted


[list]
[*] list item 1
[*] list item 2
[*] list item 3
[/list]

Numbered


[list=1]
[*] list item 1
[*] list item 2
[*] list item 3
[/list]
Blockquote 

[quote]Quoted text[/quote]
Code 

[code]
echo 'Hello World';
[/code]
Text color

Three or more dashes or asterisks:

[color=red]Red colored text[/color]
[color=#000000]Black colored text[/color]
Text size 
[size=150]Text with 150% size[/size]
[size=200]Text with 200% size[/size]
